# 学习笔记

## 位运算

- 异或：不进位加法
  - x ^ 0 = x
  - x^ 1s(全1，0取反 ~0)
  - x & ~x = 1s
  - x^x=0
  - swap：c = a ^ b => a ^ c = b, b ^ c  = a 
  - a ^ b ^ c  = (a ^ b) ^ c = a ^ (b ^ c)
- 实际应用
  - 判断奇偶
    - 奇数:  x % 2 = 1 -> x & 1 = 1
    - 偶数：x % 2 = 0 -> x & 1 = 0
  - 判断二的n次幂
    - x & ( x  - 1) == 0
  - 中位数
    - mid = (left + right) / 2 -> mid = (left + right) >> 1
  - 去最低位1
    - x = x & (x - 1)
  - 得到最低位的1
    - x & -x
  - x & ~x = 0
